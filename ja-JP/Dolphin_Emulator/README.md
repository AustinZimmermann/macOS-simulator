### ⌨️ Dolphinエミュレータの操作（疑似コマンド）

次に実行したいアクションをテキストで入力してください。画面が切り替わります。

* **「大乱闘スマッシュブラザーズを起動して」** -> ゲームの実行画面（アスキーアート風）に切り替わります。
* **「グラフィック設定を開いて」** -> 解像度（内部解像度 $4\times$ Native など）やバックエンド（Vulkan / Metal）の設定画面を開きます。
* **「コントローラーのキー配置を変えたい」** -> キーボードや外部ゲームパッドのマッピング画面を表示します。
* **「新しいISO（ゲーム）を追加して」** -> デスクトップにある他のゲームファイルを読み込みます。
