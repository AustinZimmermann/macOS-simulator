### ⌨️ リモコン・コントロール（疑似操作）

VLCプレイヤーをどう操作しますか？キーボードショートカットやアクションを選んで教えてください。

* **「`Space`（スペースキー）を押す」** ── 動画を一時停止（⏸️）または再生（▶️）します。
* **「`⌘ + ↑` / `⌘ + ↓` を押す」** ── 音量を上げ下げします。
* **「`→` / `←` を押す」** ── 10秒早送り、または巻き戻しをします。
* **「字幕ファイルをドロップする」** ── 字幕（.srt）を読み込んで画面に表示します。
* **「イコライザーを [ Rock ] に変更」** ── 音質プリセットを切り替えます。
